# Stronghold Tester

A Fabric mod for Minecraft 1.16.1 to test stronghold navigation on newly created worlds.

The player is spawned as close to the stronghold as possible. There is also a nether portal to indicate where you would spawn from if portaling from the nether with the exact coordinates of the stronghold.

The player is also given a loadout (e.g. ender eyes, iron gear) to assist with navigation.
